Before you start the program, please make sure :

1. you had your rawData oganized as EXPname/rawData and with Mosic.txt file in the EXPname folder.

2. you checked the stitchitConf.ini file parameters, such as 20x zeiss or Nikon 16x objective, scan range 3.2 or 2.0, fit your experiments.

3. make sure your ch1 is red, ch2 green, ch3 is blue, otherwise you need to change as this sequence after stitching.

4. you have download all the script files and added to your matlab pathway
